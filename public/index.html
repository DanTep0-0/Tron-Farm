Q&A

Is it safe?

This game is based on the smart-contract Tron Network. No one can affect the content of the contract because it is decentralized. Smart-contract is an open sourse so it is available for anyone to study.

How to start earning real money?

If you want to get profit, you need to buy one or more animals for an in-game currency - coins.

What are coins?

Tron Farm coins are the in-game currency. They are used for purchasing animals or improving your animals' nutrition. You get profit in coins too.

Coin rate is fixed: 1TRX = 80 Coins.

What do I need to start playing?

For the game to work and to conduct transactions to the smart-contract address properly, you have to install the TronLink extension in your browser and to sign up / log in.

Make sure you are connected to the Mainnet, not to Shasta (you can change it in the TronLink settings).

For access through mobile devices you can use Yandex-browser (for Android), which supports TronLink's full work.

How to invest or withdraw money?

You can carry out these operations in the game itself in the "My Account" section. Money could be set in game currency (Coins) or in Trons (TRX).

How do I get profit?

Profit from your animals is calculated and added automatically every hour after the first investment. You make profit ONLY as long as there are money in the smart-contract.

What is your fee amount?

The administration fee equals 10% of the coins you spent for the animals purchase.
